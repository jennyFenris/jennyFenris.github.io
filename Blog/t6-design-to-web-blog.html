What a responsive site is and why responsiveness is important?

A responsive website is a site that reshape/resizes its layout depending on the device or browser size it is being seen through. Responsive websites are important because of the growth of smartphones and tablets and other mobile devices. More people are using smaller screen devices to view Web pages.

What is mobile first design and why is it important?

In terms of mobile web design, this means that a full, standard website would scale back and gradually remove content and features as the viewport became smaller and the system simpler. It is important because it allows for easier/better experience when web browsing on a smaller device.

What is a wireframe and why do we use it?

A website wireframe is a skeletal framework of a website. It allows us to arrange Elements(boxes) in a way we see fit. Wireframe layout allows us to semantically structure our html/css
