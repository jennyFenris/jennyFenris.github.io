Can you think of an anology to describe HTML, CSS, and the DOM? How would you describe it to your non-technical friend?

  • Html stands for "Hype Text Markup Language". It is a set of rules that tell the computers how to interpret code to display websites".
  • CSS stands for "Cascading Style Sheets". It is a set of rules that tell the computer how to present a website(visually)
  • DOM stands for "Document Object Model". It is the internal representation of the page in a hierachical structure of nodes. It is best to think of it like a tree. html tags branching Head and body tags

What is meant by boxifying design?

Boxifying is the concept that each element is contained within a box. Shifiting rearranging those boxes makes it easier to organise, design and visualise a webpage or follow a given website design. boxes can be defined by a group of element/s using the div tag.

What is the box model?

In CSS the term box model is used when talking about the design and layout. The CSS box model is essentialy a box that wraps around every HTML element. It consists of : margins, borders, padding, and the actual content.
